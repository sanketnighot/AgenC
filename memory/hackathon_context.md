## Event

**ETHGlobal Open Agents** — async hackathon, April 24 – May 6, 2026. Build from anywhere.

## Gensyn Prize: $5,000 — "Best Application of AXL"

| Place | Amount |
|-------|--------|
| 1st | $2,500 |
| 2nd | $1,500 |
| 3rd | $1,000 |

**Judging criteria (in order of importance):**
1. Depth of AXL integration
2. Quality of code
3. Clear documentation
4. Working examples

**Hard requirements (disqualifying if missing):**
1. Must use AXL for inter-agent or inter-node communication — no centralized message broker replacing AXL
2. Must demonstrate communication across **separate** AXL nodes, not just in-process

**Bonus:** Winners are fast-tracked into the Gensyn Foundation grant programme.

## Winning Strategy

Focus on **Phase 4** (competitive bidding + visual node status) because:
- "Depth of AXL integration" rewards showing workers race/negotiate over the mesh
- "Working examples" rewards the live demo showing 3+ nodes active simultaneously
- A visual node map makes the P2P nature undeniable to judges

**Why:** The hackathon ends May 6, 2026 — very close to current date (May 1, 2026). 5 days left.

**How to apply:** Every dev decision should maximize demo impact. Working > perfect.
